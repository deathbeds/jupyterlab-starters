# jupyterlab-starters

> _Parameterized file and directory starters for JupyterLab._

[![badge-binder][]][binder] [![badge-ci][]][ci]

## What's a starter?

- [x] A single file
- [x] A directory
- [x] A python function
- [x] A [cookiecutter][]
- [ ] A shell script

... that shows up in your JupyterLab at the click of a button
